INSTALL (one time, from the project root)

The key is read from the environment and written to .hermes/.env, which is
gitignored. Never put the real key in this file — GitHub push protection
blocks it, and it would be public the moment it lands.

  PowerShell:
      $env:DEEPSEEK_API_KEY="sk-your-key"; bash deploy/install.sh

  bash (Linux / macOS / Git Bash):
      DEEPSEEK_API_KEY=sk-your-key bash deploy/install.sh


RUN

  Windows (PowerShell / cmd):
      .\hermes.cmd

  Linux / macOS / Git Bash:
      ./hermes
